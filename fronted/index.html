MoodTrack

Twitter has been used as real time source for applications.Every day,on average,around 500 millions tweets are tweeted. This amount of data, lead me to the idea of creating a real time application that displays some of the tweets on google map using a sentiment tool(module).MoodTrack displays tweets,using emoticons(positive,neutral,negative) which depends from tweet's content.

The system filters the Tweets from the Twitter App Stream. Moreover, only geolocated tweets and tweets that their geolocation is extracted from the userâ€™s profile location are being displayed.

New tweets are pinned in a worldwide map,with geolocation info,which are extracted from the user's profile. Visuals are displayed on a styled Google map,and if you want you can see the content of the tweet.
